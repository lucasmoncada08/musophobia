# Keyboard Commands

Press `?` on any page to show the help menu with all available commands.

## Navigation

| Key | Description |
|-----|-------------|
| `j` | Scroll down (hold for continuous) |
| `k` | Scroll up (hold for continuous) |
| `h` | Scroll left (hold for continuous) |
| `l` | Scroll right (hold for continuous) |
| `d` | Half page down |
| `u` | Half page up |
| `gg` | Go to top of page |
| `G` | Go to bottom of page |

## Help

| Key | Description |
|-----|-------------|
| `?` | Show/hide help menu |

## Notes

- Commands are disabled when focus is on input fields, textareas, or contenteditable elements
- Help menu can be dismissed with `?`, `Escape`, or by clicking outside the modal
