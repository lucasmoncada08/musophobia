# Keyboard Commands

Press `?` on any page to show the help menu with all available commands.

## Navigation

| Key | Description |
|-----|-------------|
| `j` | Scroll down (hold for continuous) |
| `k` | Scroll up (hold for continuous) |
| `h` | Scroll left (hold for continuous) |
| `l` | Scroll right (hold for continuous) |
| `d` | Half page down |
| `u` | Half page up |
| `gg` | Go to top of page |
| `G` | Go to bottom of page |

## Links

| Key | Description |
|-----|-------------|
| `f` | Show link hints (click in current tab) |
| `F` | Show link hints (open in new tab) |

### During Link Hint Mode

| Key | Description |
|-----|-------------|
| `s/a/d/f/j/k/l` | Type hint characters to select element |
| `Escape` | Cancel hint mode |
| `Backspace` | Delete last typed character |

## Help

| Key | Description |
|-----|-------------|
| `?` | Show/hide help menu |

## Notes

- Commands are disabled when focus is on input fields, textareas, or contenteditable elements
- Press `Escape` while in an input field to blur it and resume keyboard navigation
- Help menu can be dismissed with `?`, `Escape`, or by clicking outside the modal
- Link hints show labels on clickable elements (links, buttons, inputs) in the viewport
- Text inputs are focused instead of clicked; buttons and links are clicked
- In new tab mode (`F`), links open in a new tab while inputs are focused normally
